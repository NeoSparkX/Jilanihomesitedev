> **Jilani Home** is a premium, high-performance real estate marketplace designed to connect users directly with verified office spaces, event halls, and residential properties.

---

## ✨ Features

- 💎 **Liquid Glass Aesthetic**: A state-of-the-art UI featuring high-blur backdrop filters, glossy gradients, and frosted glass components.
- 📱 **Mobile First**: Fully responsive layouts optimized for narrow screens with ergonomic navigation.
- 🌓 **Smart Theming**: Intelligent Light and Dark mode support with high-contrast accessibility.
- ⚡ **Direct Connectivity**: Skip the middleman with verified host contacts and direct negotiation features.
- 🗺️ **Verified Listings**: Curated selection of premium properties in prime locations.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [pnpm](https://pnpm.io/)

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:5173`.

## 🏗️ Architecture

The project is structured for scalability and performance:

- `src/app/components/`: Reusable, atomic UI components (Navbar, Hero, Listings, etc.).
- `src/app/pages/`: Main view components mapped to routes.
- `src/styles/`: Global styles and theme configurations using Tailwind v4.
- `src/app/theme-context.tsx`: Context-based theme management.
- `docs/`: Comprehensive project documentation and guidelines.
